O Trabalho foi feito em Python 3.9, então é nescessário python 3 para executa-lo.
Para executar o programa basta entrar no diretorio do arquivo escrever no cmd: "python trabSC.py".

Intruções de uso:
O programa possui um menu com 4 opções, para escolhe-las basta digitar o número da respectiva função e teclar enter:
1 -> pede uma frase para ser cifrada e uma senha para ser untilizada no processo, retornando a frase cifrada 
2 -> pede uma frase cifrada para ser decifrada e uma senha para ser untilizada no processo, retornando a frase decifrada 
3 -> pede uma frase cifrada para descobrir a senha dessa frase utilizando frequencias de letras, retornando a senha e a frase decifrada 
0 -> termina o programa
